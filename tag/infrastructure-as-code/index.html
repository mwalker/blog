Auth0 BrandingTheme in pulumi

Updating to Universal Login on Auth0 and we need to access what pulumi calls the BrandingTheme accessible in the console from “Branding > Universal Login > Customization Options”. To import them using pulumi requires an id, and it turns out that id is default. Thanks to karolis.mazukna on the Auth0 Community.
